## 📖 基于碰撞预测的2D弹球打砖块
- 纯运动学的完全弹性碰撞，没有用到任何物理系统
- 解析法预测碰撞，而非牛顿法。速度再快也不会穿模
- 随机生成7种预设的多边形，可以自己添加预制体和顶点信息当作新的多边形
- 直接运行ball场景即可，小球停下后，再次点击左键会朝紫色方块发射
